I mentioned in my November 2025 recap that I found a bitmap font editor: BITFONTMAKER2.

I immediately wanted to make something with it. I've dabbled with other font editors but nothing comes close to how simple this is.

So here it is:

JBLOCK!

Go to a full font-demo.

JBLOCK is a 14x14 pixel, uppercase only, monospace font.

The intention when designing this was to make each character occupy as much space as possible while still being recognisable. I started off with a 13x10 pixel rectangle in the editor and deleted pixels to create each glyph.

There are a number of technical and practical limitations with this font:

  • It only has uppercase characters, lowercase letters are duplicates of the uppercase
  • There are no ascenders or descenders, with the exception of a couple of special characters that creep slightly past the baseline and x-height
  • Only basic Latin characters are supported
  • I'm not satisfied with the numerals, it's not clear that some of them aren't letters. Specifically '2' looks like 'Z'
  • Very illegible and not suitable for longer blocks of text. Minimum readable size is about 24px

But above all, it's a pain to read.

This was a fun little challenge for myself. I have no intention to actually use this for anything, but feel free to download the .ttf if you want it.

Side-quest within a side-quest: I made a little script to demo fonts when making this page. This is what you see at the bottom of this page.

You simply tell the script what container to append elements to, the name of your font-family and the base font-size.

The demo script takes care of the rest, creating elements to showcase the font at different sizes, the full character set and how the font reads in a bigger block.
